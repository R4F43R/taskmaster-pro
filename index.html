TaskMaster Pro

Instrucciones de Despliegue

GitHub Pages solo puede servir contenido estático. Para ejecutar la aplicación completa:

1

Desplegar el Backend en Render

El backend (Node.js + SQLite) debe desplegarse en Render.com

# Crear nuevo Web Service en Render
# Conectar repositorio de GitHub
# Configurar:
  - Runtime: Node
  - Build Command: npm install
  - Start Command: npm start
  - Plan: Free
# Agregar variables de entorno
  JWT_SECRET=tu_secreto_jwt
  DB_PATH=/data/taskmaster.db
# Agregar disco persistente
  - Path: /data
  - Size: 1GB
# Crear servicio
2

Configurar el Frontend

Actualizar la URL del backend en js/auth.js:

// Cambiar esta línea:
const API_BASE_URL = window.location.origin;

// Por la URL de tu backend en Render:
const API_BASE_URL = "https://taskmaster-backend-2cu9.onrender.com";
3

Desplegar Frontend en GitHub Pages

Configurar GitHub Pages para servir la carpeta /public

1. Ir a Settings > Pages en tu repositorio
2. En "Source", seleccionar "Deploy from a branch"
3. Branch: main (o tu rama principal)
4. Folder: /public
5. Guardar
4

Probar la Aplicación

Visita tu GitHub Pages URL para usar la aplicación

El backend se ejecutará en Render y el frontend en GitHub Pages
